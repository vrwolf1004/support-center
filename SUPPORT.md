# 지원 안내

문제 발생 시 아래 정보를 포함해 이슈를 생성해 주세요.

필수 정보:
- 확장 버전(예: v0.5.0)
- Chrome 버전(예: 114.0.5735.199)
- 운영체제(예: Windows 10)
- 재현 단계(순서대로)
- 콘솔 에러(가능하면 복사/붙여넣기)

1. 이슈 생성(템플릿 사용)
2. 담당자 확인 후 답변
3. 필요 시 추가 정보 요청
