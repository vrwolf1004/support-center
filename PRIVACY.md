# 개인정보 처리방침 (요약)

이 확장 프로그램은 개인식별정보(PII)를 수집하지 않습니다.

수집 항목 및 목적:
- 로컬에 저장된 설정(config) — 사용자 환경 유지
- 사용자가 입력한 API 키(사용자 소유) — 외부 API 호출 시 사용(로컬 보관)

보관 및 제공:
- 저장 위치: `chrome.storage.local` (로컬)
- 제3자 제공: 없음
- 보관 기간: 사용자가 삭제할 때까지

보안 권고:
- 민감 데이터는 가능하면 저장하지 마십시오. 저장 시 암호화 권장
