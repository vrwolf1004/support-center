# StockPulse 개인정보 처리 방침

- 수집 항목: 제품 동작에 필요한 최소한의 진단 정보(앱 버전, 오류 로그, 선택적 사용 통계).
- 개인 식별 정보(이메일 등)는 사용자가 자발적으로 제공하지 않는 한 수집하지 않습니다.
- 수집된 데이터는 문제 해결과 서비스 개선에만 사용됩니다.
- 외부 제공: 법적 요청 또는 사용자의 명시적 동의가 있는 경우에만 제한적으로 공유합니다.

문의: 지원 관련은 Issues 탭( https://github.com/vrwolf1004/support-center/issues )을 이용해 주세요.
