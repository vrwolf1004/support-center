# Help Me Hide

선택한 텍스트를 PIN으로 암호화하여 숨기는 Chrome 확장입니다.

## 설치

1. 확장을 설치합니다.
2. 툴바에 Help Me Hide 아이콘이 나타나면 설치 완료입니다.

## 사용법

### PIN 설정

1. 툴바의 Help Me Hide 아이콘을 클릭합니다.
2. **설정** 탭을 선택합니다.
3. 4자리 PIN 번호를 입력합니다 (예: `1234`).
4. **저장** 버튼을 클릭합니다.

### 텍스트 암호화

1. 웹페이지에서 텍스트를 선택합니다.
2. 우클릭 → **"선택한 텍스트 숨기기"** 선택합니다.
3. 암호화된 텍스트가 자동으로 클립보드에 복사됩니다.

### 텍스트 복호화

1. 암호화된 텍스트를 선택합니다 (마커 포함: `<!--ENC:V1:...-->`).
2. 우클릭 → **"복원"** 선택합니다.
3. PIN 입력 프롬프트에 설정한 PIN을 입력합니다.
4. 원본 텍스트가 복원되어 클립보드에 복사됩니다.

## 기능

| 기능 | 설명 |
|------|------|
| 🔐 AES-GCM 암호화 | 강력한 암호화 방식으로 텍스트 보호 |
| 🔑 PIN 기반 보안 | 4자리 PIN으로 간단하게 관리 |
| 📋 자동 복사 | 암호화된 텍스트 자동 클립보드 복사 |
| ⚡ 빠른 모드 | 빠른 암호화/복호화 옵션 |
| 🌐 모든 사이트 지원 | 웹의 어디서나 사용 가능 |

## 설정 옵션

### 암호화 모드

1. **설정** 탭에서 암호화 모드를 선택합니다:
   - **AES-GCM** (권장): 강력한 보안
   - **Simple** : 빠른 처리

### 마커 형식

1. **마커 형식** 옵션에서 선택합니다:
   - **HTML 주석**: `<!--ENC:V1:...-->`
   - **괄호**: `[ENC:V1:...]`

## 보안 주의사항

- **4자리 PIN은 약한 보안입니다** (10,000가지 조합만 가능)
- **민감한 정보는 저장하지 마세요** (비밀번호, 금융정보 등)
- PIN을 다른 사람과 공유하지 마세요
- 이 확장은 **교육 및 개념 검증 목적**으로 설계되었습니다
- 실제 강력한 보안이 필요하면 전문 암호화 도구를 사용하세요

## 업데이트

- Chrome 확장 프로그램은 자동으로 업데이트됩니다.
- 수동: `chrome://extensions` → **업데이트** 클릭
