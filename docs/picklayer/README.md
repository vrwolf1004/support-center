# PickLayer

> 개발자와 디자이너가 빠르게 고품질의 사용자 인터페이스를 만들 수 있도록 지원하는 크로스플랫폼 디자인 시스템

## 📖 개요

**PickLayer.net**은 다양한 스타일과 테마를 지원하는 UI 컴포넌트 라이브러리입니다. 일관된 디자인 언어를 통해 사용자 경험을 향상시키고, 빠른 개발을 가능하게 합니다.

## 🎨 주요 특징

### 테마
- **12개의 다양한 테마** — Basic, Dark, Flat, Minimalism 등
- **완전한 커스터마이징** — 모든 테마는 MIT 라이선스로 자유롭게 사용 가능

### 다국어 지원
- **10개 언어** 기본 지원
- 글로벌 사용자 대응

### 반응형 디자인
- 모바일부터 데스크탑까지 **완전 대응**
- 모든 기기에서 일관된 경험

### 실시간 댓글 시스템
- **Firebase 기반** 실시간 익명 댓글
- 10개 언어 기반 **자동 필터링**

## 🛠️ 기술 스택

| 분야 | 기술 |
|------|------|
| **Frontend** | HTML5, CSS3, Vanilla JavaScript |
| **Backend** | Firebase (인증, Firestore) |
| **호스팅** | Cloudflare Pages, Firebase Hosting |
| **다국어** | i18n (10개 언어) |

## 📜 라이선스

모든 테마는 **MIT 라이선스**로 개인 및 상업 프로젝트에 자유롭게 사용 가능합니다.

> 모든 테마는 AI(GPT-4)의 도움으로 생성된 오리지널 디자인으로, 저작권 문제가 없습니다.
