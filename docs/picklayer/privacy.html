PickLayer 개인정보 처리방침

최종 수정일: 2026년 3월 28일

1. 개인정보의 수집 및 이용

수집 항목

  • 사용자 기기 정보 (IP 주소, 기기 ID, 사용자 에이전트)
  • 쿠키 및 로컬 스토리지 데이터
  • 페이지 접속 기록 및 행동 분석

수집 목적

  • 서비스 개선
  • 사용 통계 분석
  • 광고 제공

2. Google AdSense

본 사이트는 Google AdSense를 통해 광고를 게재하며, Google은 다음과 같이 개인정보를 처리합니다:

  • 관심사 기반 광고 제공
  • 쿠키를 통한 사용자 추적
  • Google 프라이버시 정책에 따른 데이터 수집
Google 프라이버시 정책: https://policies.google.com/privacy

3. 쿠키 사용

사용자 경험 개선 및 분석을 위해 쿠키를 사용하며, 브라우저 설정에서 비활성화 가능합니다.

4. 데이터 보안

사용자 정보는 HTTPS를 통해 암호화되어 전송됩니다.

5. 댓글 콘텐츠 필터링

부적절한 표현을 포함한 댓글은 자동으로 차단되며, 10개 언어 기반의 단어 목록과 패턴 규칙으로 운영됩니다.

6. 문의

  • GitHub Issues: 버그, 기능 요청, 피드백
  • 이메일: 개인/보안 관련 문의
본 개인정보 처리방침은 사전 공지 없이 변경될 수 있으며, 변경 시 본 페이지를 통해 공지됩니다.
