# 1Hour1Word

Learn a new foreign language word every hour with notifications in this Chrome extension.

## Installation

1. Search for "1Hour1Word" on the Chrome Web Store.
2. Click "Add to Chrome".
3. Installation is complete when the 1Hour1Word icon appears in the toolbar.

## Usage

### Word Learning

1. Click the 1Hour1Word icon in the toolbar.
2. Select a language (English / Spanish / Japanese).
3. Click the 3D card to flip it and check the meaning.
4. Click the **Save** button to add the word to your vocabulary list.

### Web Page Search

Select a word or sentence on a web page and an automatic popup will appear.
- Word: Check meaning, pronunciation, and examples
- Sentence: Korean translation

## Settings (Options)

Click the **⚙️ Settings** button at the bottom of the popup to open options.

| Item | Description | Default |
|------|-------------|---------|
| Language | Select the language to learn | English |
| Notification Interval | Frequency of word notifications (minutes) | 60 minutes |
| Dark Mode | Apply dark theme | Disabled |

## Updates

- Chrome extensions update automatically.
- Manual: `chrome://extensions` → Click **Update**
