# 1Hour1Word 개인정보 처리방침

최종 수정일: 2026-04-23

## 1. 수집하는 정보

1Hour1Word는 서비스 제공을 위해 다음 정보를 처리합니다.

| 구분 | 항목 | 목적 | 저장 위치 |
|------|------|------|----------|
| 자동 수집 | 학습 통계 (단어 클릭 횟수, 저장 여부) | 학습 진도 추적 | 사용자 기기 |
| 선택 수집 | 개인 단어장 (저장한 단어 목록) | 학습 콘텐츠 | 사용자 기기 |
| 선택 수집 | 설정 (언어, 알림 간격, 다크 모드) | 사용자 경험 개선 | 사용자 기기 |

- **개인 식별 정보(이름, 이메일, 계정 정보 등)는 수집하지 않습니다.**
- **수집된 모든 데이터는 사용자의 기기에만 저장되며, 외부 서버로 전송되지 않습니다.**

## 2. 외부 API 사용

### Glosbe API
- **목적**: 단어의 뜻, 발음, 예문 조회
- **데이터 전송**: 조회하는 단어만 Glosbe 서버로 전송
- **개인정보 포함 여부**: 없음 (단어 데이터만 전송)
- **보관 기간**: Glosbe의 개인정보 처리방침에 따름

## 3. 정보의 이용 목적

- 개인 단어장 관리 및 학습 진도 추적
- 사용자 설정(언어, 알림 간격) 저장
- 서비스 기능 제공 (단어 알림, 웹페이지 검색)

## 4. 정보의 보관 및 파기

- **로컬 저장소**: 사용자가 확장을 삭제할 때까지 기기에 보관
- **자동 파기**: 확장 프로그램 삭제 시 IndexedDB에 저장된 모든 데이터 자동 삭제
- **수동 삭제**: 사용자는 언제든지 Chrome 저장소를 초기화하여 모든 데이터 삭제 가능

## 5. 사용자의 권리

- **언제든지 확장 삭제**: 확장을 제거하면 모든 데이터가 자동으로 삭제됩니다
- **데이터 직접 관리**: 저장된 모든 데이터는 사용자 기기 내에만 존재하므로, 사용자가 직접 제어합니다
- **개인정보 열람/삭제 요청**: 불가능 (외부 서버에 저장되지 않음)

## 6. 보안

- Chrome 확장의 sandbox 환경으로 보호됨
- HTTPS를 통한 Glosbe API 통신 (암호화)
- 서드파티 서버로 개인 데이터 전송 없음

## 7. 기술 정보

**로컬 저장 메커니즘**:
- IndexedDB: 개인 단어장, 학습 통계
- Chrome Storage API: 사용자 설정

**전송되는 데이터**:
- Glosbe API 요청 시: 조회하는 단어만 전송
- 분석 데이터: 없음

## 8. 정책 변경

개인정보 처리방침은 서비스 개선을 위해 변경될 수 있습니다.
변경 시 확장의 업데이트 공지를 통해 안내됩니다.
