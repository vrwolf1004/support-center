# 1Hour1Word Privacy Policy

Last Updated: 2026-04-23

## 1. Information We Collect

1Hour1Word processes the following information to provide the service.

| Category | Items | Purpose | Storage Location |
|----------|-------|---------|-----------------|
| Auto-collected | Learning statistics (word clicks, saves) | Track learning progress | User's device |
| User-provided | Personal vocabulary (saved words) | Learning content | User's device |
| User-provided | Settings (language, notification interval, dark mode) | User experience | User's device |

- **We do not collect personally identifiable information (name, email, account information, etc.).**
- **All collected data is stored only on the user's device and is not transmitted to external servers.**

## 2. External API Usage

### Glosbe API
- **Purpose**: Look up word meanings, pronunciations, and example sentences
- **Data Transmission**: Only the queried word is sent to Glosbe servers
- **Personal Information**: None (only word data is transmitted)
- **Retention**: According to Glosbe's Privacy Policy

## 3. Purposes of Information Use

- Manage personal vocabulary and track learning progress
- Save user settings (language, notification interval)
- Provide service features (word notifications, webpage search)

## 4. Data Retention and Deletion

- **Local Storage**: Retained on device until user uninstalls the extension
- **Automatic Deletion**: All IndexedDB data is automatically deleted when the extension is removed
- **Manual Deletion**: Users can manually clear Chrome storage to delete all data at any time

## 5. User Rights

- **Uninstall at Any Time**: Removing the extension automatically deletes all data
- **Direct Data Control**: All stored data exists only on the user's device, giving you full control
- **Data Access/Deletion Requests**: Not applicable (data is not stored on external servers)

## 6. Security

- Protected by Chrome extension's sandbox environment
- Glosbe API communication via HTTPS (encrypted)
- No personal data transmitted to third-party servers

## 7. Technical Information

**Local Storage Mechanisms**:
- IndexedDB: Personal vocabulary, learning statistics
- Chrome Storage API: User settings

**Transmitted Data**:
- Glosbe API requests: Only the queried word is sent
- Analytics: None

## 8. Policy Changes

This Privacy Policy may be updated to improve the service.
Updates will be notified through the extension's update notices.
