# 1Hour1Word

매 시간 한 단어씩, 알림으로 배우는 외국어 학습 Chrome 확장입니다.

## 설치

1. [Chrome 웹 스토어](#)에서 "1Hour1Word"를 검색합니다.
2. **Chrome에 추가**를 클릭합니다.
3. 툴바에 1Hour1Word 아이콘이 나타나면 설치 완료입니다.

## 주요 기능

### 🌍 다중 언어 지원

- 🇬🇧 **영어** — English
- 🇪🇸 **스페인어** — Español  
- 🇯🇵 **일본어** — 日本語

### 📡 1시간마다 자동 알림

- Chrome 알람으로 설정한 간격(기본 60분)마다 새로운 단어 전송
- 브라우저 팝업 알림으로 단어 표시
- 알림 간격은 설정에서 자유롭게 조정 가능

### 3D 플립 카드

- **앞면**: 단어 + 발음
- **뒷면**: 의미 + 예문  
- 마우스 호버 또는 터치로 카드 뒤집기

### 💾 로컬 저장

- **IndexedDB**: 단어와 학습 진도 기기 내 저장
- 인터넷 연결 없이 오프라인 학습 가능

### 📊 학습 통계

- 총 학습한 단어 수
- 연속 학습 일수
- 언어별 학습 분포

## 사용법

### 1단계: 언어 선택

1. 툴바의 1Hour1Word 아이콘을 클릭합니다.
2. 언어 칩(영어 / 스페인어 / 일본어)을 선택합니다.
3. 현재 단어가 팝업에 표시됩니다.

### 2단계: 단어 학습

팝업의 **3D 카드**에서:
- 단어와 발음을 확인합니다
- **마우스 호버** 또는 **클릭**하여 카드를 뒤집습니다
- 의미와 예문을 학습합니다

### 3단계: 단어장 저장

- **저장** 버튼을 누르면 해당 단어를 개인 단어장에 추가합니다
- **다음 단어** 버튼으로 다른 단어를 확인할 수 있습니다

### 4단계: 웹페이지에서 검색

웹페이지 상의 텍스트를 선택하면:
- **단어**: 뜻, 발음, 예문 팝업
- **문장**: 한국어 번역 팝업

## 설정 (Options)

팝업 하단의 **⚙️ 설정**을 클릭하면 상세 옵션이 열립니다.

| 항목 | 설명 | 기본값 |
|------|------|--------|
| **언어** | 학습할 언어 선택 | 영어 |
| **알림 간격** | 단어 알림 빈도(분) | 60분 |
| **자동 음성** | 단어 발음 자동 재생 | 비활성 |
| **다크 모드** | 다크 테마 적용 | 비활성 |
| **API 제공자** | 의미 조회 소스 선택 | 자동 |

### 백업 및 복원

- **다운로드**: 현재 단어장을 JSON 파일로 저장
- **업로드**: JSON 파일에서 단어장 복원

## 웹페이지 텍스트 검색

웹페이지 어디서든 단어를 선택하면 자동 팝업이 표시됩니다.

### 지원되는 검색 방식

| 언어 | 조회 방식 |
|------|--------|
| **영어** | Free Dictionary API → Google Translate |
| **다른 언어** | Google Translate → MyMemory |

> **팁**: 고유명사(도시, 인명)는 한국어 번역으로 표시됩니다.

## 업데이트

- Chrome 확장 프로그램은 **자동으로 업데이트**됩니다
- 수동 확인: `chrome://extensions` → **업데이트** 클릭

## 시스템 요구사항

- Chrome 88 이상
- 활성 인터넷 연결 (의미 조회 시)
- 최소 저장 공간: 10MB (IndexedDB)
