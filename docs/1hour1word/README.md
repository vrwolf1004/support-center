# 1Hour1Word

매 시간 한 단어씩, 알림으로 배우는 외국어 학습 Chrome 확장입니다.

## 설치

1. Chrome 웹 스토어에서 "1Hour1Word"를 검색합니다.
2. Chrome에 추가를 클릭합니다.
3. 툴바에 1Hour1Word 아이콘이 나타나면 설치 완료입니다.

## 사용법

### 단어 학습

1. 툴바의 1Hour1Word 아이콘을 클릭합니다.
2. 언어(영어 / 스페인어 / 일본어)를 선택합니다.
3. 3D 카드를 클릭하여 뒤집고 의미를 확인합니다.
4. **저장** 버튼으로 단어장에 추가합니다.

### 웹페이지 검색

웹페이지에서 단어나 문장을 선택하면 자동 팝업이 표시됩니다.
- 단어: 뜻, 발음, 예문 조회
- 문장: 한국어 번역

## 설정 (Options)

팝업 하단의 **⚙️ 설정**을 클릭하면 옵션이 열립니다.

| 항목 | 설명 | 기본값 |
|------|------|--------|
| 언어 | 학습할 언어 선택 | 영어 |
| 알림 간격 | 단어 알림 빈도(분) | 60분 |
| 다크 모드 | 다크 테마 적용 | 비활성 |

## 업데이트

- Chrome 확장 프로그램은 자동으로 업데이트됩니다.
- 수동: `chrome://extensions` → **업데이트** 클릭
