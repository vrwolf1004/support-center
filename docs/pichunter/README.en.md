# PicHunter

A Chrome extension that scans webpages for images and helps you download them easily.

## Installation

1. Install the extension.
2. Installation is complete when the PicHunter icon appears in the toolbar.

## Usage

### Scan Images

1. Open the webpage where you want to download images from.
2. Click the PicHunter icon in the toolbar.
3. Click the **Scan** button.
4. Downloadable images will be highlighted on the page.

### Download Individual Images

1. Hover over a highlighted image.
2. Click the **Download** button.
3. The image will be downloaded.

### Bulk Download

1. After scanning, click the **Bulk Download** button.
2. Confirm the images you want to download in the modal.
3. Click the **Confirm** button to download all images at once.

## Features

| Feature | Description |
|---------|-------------|
| Image Scanning | Automatically detect all images on a webpage |
| Image Highlighting | Visualize downloadable images |
| Quick Download | Quick download button for individual images |
| Bulk Download | Download multiple images at once |
| Multilingual Support | Korean, English, Japanese, Spanish, French, Russian |
| Minimum Size Filtering | Select only images above desired dimensions |
| Background Image Detection | Can detect CSS background images (optional) |

## Settings

### Minimum Image Size

1. Click the PicHunter icon.
2. Select the **Settings** tab.
3. Set the minimum width and minimum height.
4. Click the **Save** button.

### Background Image Detection

1. In the **Settings** tab, check the "Background Image Detection" option.
2. Changes are saved automatically.

## Precautions

- Chrome internal pages, Web Store pages, and extension pages cannot be scanned.
- Some websites may block original image access depending on their site policy.
- Use copyrighted images for personal use only.

## Updates

- Chrome extensions update automatically.
- Manual: `chrome://extensions` → Click **Update**
