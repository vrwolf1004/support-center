# PicHunter 개인정보 처리방침

최종 수정일: 2026-04-23

## 1. 수집하는 정보

PicHunter는 서비스 제공을 위해 다음 정보를 처리합니다.

| 구분 | 항목 | 목적 | 저장 위치 |
|------|------|------|----------|
| 자동 수집 | 스캔 이력 (URL, 타임스탬프) | 사용자 편의성 | 사용자 기기 |
| 사용자 설정 | 최소 이미지 크기 필터 | 다운로드 옵션 | 사용자 기기 |
| 사용자 설정 | 배경 이미지 감지 여부 | 기능 설정 | 사용자 기기 |

- **개인 식별 정보(이름, 이메일, 계정 정보 등)는 수집하지 않습니다.**
- **다운로드한 이미지 파일은 사용자의 기본 다운로드 폴더에만 저장됩니다.**
- **방문한 웹페이지 정보는 사용자 기기에만 저장되며, 외부로 전송되지 않습니다.**

## 2. 정보의 이용 목적

- 웹페이지 이미지 검색 및 하이라이트 기능
- 빠른 다운로드를 위한 설정 저장
- 사용자 편의성 향상

## 3. 정보의 제3자 제공

수집된 정보는 원칙적으로 제3자에게 제공하지 않습니다.

**예외**:
- 법령에 따른 요청이 있는 경우
- 사용자의 명시적 동의가 있는 경우

## 4. 정보의 보관 및 파기

- **로컬 저장소**: 사용자가 확장을 삭제할 때까지 기기에 보관
- **스캔 이력**: Chrome Storage에 저장, 확장 삭제 시 함께 삭제
- **자동 파기**: 확장 프로그램 삭제 시 저장된 모든 설정 및 이력 자동 삭제
- **다운로드 파일**: 사용자가 수동으로 관리 (OS 파일 시스템)

## 5. 사용자의 권리

- **언제든지 확장 삭제**: 확장을 제거하면 모든 메타데이터가 자동으로 삭제됩니다
- **설정 직접 관리**: 최소 이미지 크기, 배경 이미지 감지 설정 제어
- **다운로드 관리**: 다운로드한 이미지 파일은 사용자가 직접 관리

## 6. 보안

- Chrome 확장의 sandbox 환경으로 보호됨
- 웹페이지 콘텐츠는 HTTPS 연결로 암호화
- 방문 기록은 로컬에만 저장, 외부 전송 없음

## 7. 기술 정보

**저장 메커니즘**:
- Chrome Storage: 스캔 이력, 사용자 설정
- 운영체제 다운로드 폴더: 이미지 파일

**외부 통신**:
- 없음 (모든 처리가 로컬에서 수행)

## 8. 주의사항

- 다운로드한 이미지의 저작권은 사용자의 책임입니다
- 저작권이 있는 이미지는 개인용도로만 사용하세요

## 9. 정책 변경

개인정보 처리방침은 서비스 개선을 위해 변경될 수 있습니다.
변경 시 확장의 업데이트 공지를 통해 안내됩니다.
