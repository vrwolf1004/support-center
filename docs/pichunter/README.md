# PicHunter

웹페이지에서 이미지를 스캔하고 다운로드할 수 있는 Chrome 확장입니다.

## 설치

1. 확장을 설치합니다.
2. 툴바에 PicHunter 아이콘이 나타나면 설치 완료입니다.

## 사용법

### 이미지 스캔

1. 이미지를 다운로드하고 싶은 웹페이지를 엽니다.
2. 툴바의 PicHunter 아이콘을 클릭합니다.
3. **스캔** 버튼을 클릭합니다.
4. 다운로드 가능한 이미지들이 하이라이트됩니다.

### 개별 다운로드

1. 하이라이트된 이미지에 마우스를 올립니다.
2. **다운로드** 버튼을 클릭합니다.
3. 이미지가 다운로드됩니다.

### 일괄 다운로드

1. 스캔 후 **일괄 다운로드** 버튼을 클릭합니다.
2. 확인 모달에서 다운로드할 이미지들을 확인합니다.
3. **확인** 버튼을 클릭하면 모든 이미지가 다운로드됩니다.

## 기능

| 기능 | 설명 |
|------|------|
| 이미지 스캔 | 웹페이지의 모든 이미지 자동 감지 |
| 이미지 하이라이트 | 다운로드 가능한 이미지 시각화 |
| 빠른 다운로드 | 개별 이미지별 빠른 다운로드 버튼 |
| 일괄 다운로드 | 여러 이미지 한 번에 다운로드 |
| 다국어 지원 | 한국어, 영어, 일본어, 스페인어, 프랑스어, 러시아어 |
| 최소 크기 필터링 | 원하는 크기 이상의 이미지만 선택 |
| 배경 이미지 감지 | CSS 배경 이미지도 감지 가능 (선택) |

## 설정

### 최소 이미지 크기

1. PicHunter 아이콘을 클릭합니다.
2. **설정** 탭을 선택합니다.
3. 최소 너비, 최소 높이를 설정합니다.
4. **저장** 버튼을 클릭합니다.

### 배경 이미지 감지

1. **설정** 탭에서 "배경 이미지 감지" 옵션을 체크합니다.
2. 변경사항이 자동 저장됩니다.

## 주의사항

- Chrome 내부 페이지, Web Store 페이지, 확장 프로그램 페이지는 스캔할 수 없습니다.
- 일부 웹사이트는 사이트 정책에 따라 원본 이미지 접근을 제한할 수 있습니다.
- 저작권이 있는 이미지는 개인 용도로만 사용하세요.

## 업데이트

- Chrome 확장 프로그램은 자동으로 업데이트됩니다.
- 수동: `chrome://extensions` → **업데이트** 클릭
