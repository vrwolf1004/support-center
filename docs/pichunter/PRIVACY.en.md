# PicHunter Privacy Policy

Last Updated: 2026-04-23

## 1. Information We Collect

PicHunter processes the following information to provide the service.

| Category | Items | Purpose | Storage Location |
|----------|-------|---------|-----------------|
| Auto-collected | Scan history (URLs, timestamps) | User convenience | User's device |
| User-provided | Minimum image size filter | Download options | User's device |
| User-provided | Background image detection setting | Feature configuration | User's device |

- **We do not collect personally identifiable information (name, email, account information, etc.).**
- **Downloaded image files are stored only in the user's default download folder.**
- **Visited webpage information is stored only on the user's device and not transmitted externally.**

## 2. Purposes of Information Use

- Search and highlight images on webpages
- Save settings for quick downloads
- Improve user experience

## 3. Third-Party Data Sharing

Collected information is not provided to third parties in principle.

**Exceptions**:
- Legal requests as required by law
- Explicit user consent

## 4. Data Retention and Deletion

- **Local Storage**: Retained on device until user uninstalls the extension
- **Scan History**: Stored in Chrome Storage, deleted when extension is removed
- **Automatic Deletion**: All settings and history are automatically deleted when the extension is removed
- **Downloaded Files**: User-managed (OS file system)

## 5. User Rights

- **Uninstall at Any Time**: Removing the extension automatically deletes all metadata
- **Manage Settings**: Control minimum image size and background image detection
- **Download Management**: User manages downloaded image files directly

## 6. Security

- Protected by Chrome extension's sandbox environment
- Webpage content encrypted via HTTPS connections
- Browsing history stored locally only, no external transmission

## 7. Technical Information

**Storage Mechanisms**:
- Chrome Storage: Scan history, user settings
- Operating System Downloads Folder: Image files

**External Communication**:
- None (all processing performed locally)

## 8. Important Notes

- Users are responsible for downloaded images' copyright
- Copyrighted images should be used for personal use only

## 9. Policy Changes

This Privacy Policy may be updated to improve the service.
Updates will be notified through the extension's update notices.
