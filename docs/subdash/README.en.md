# SubDash

A Chrome extension that visually manages your monthly subscription service costs with a timeline.

## Installation

1. Install the extension.
2. Installation is complete when the SubDash icon appears in the toolbar.

## Usage

### Add Service

1. Click the SubDash icon in the toolbar.
2. Click the **+ Add** button.
3. Enter information in the right panel:
   - Service name (e.g., Claude Pro)
   - Monthly amount (in ₩)
   - Choose color
   - Subscription period (start date ~ end date)

### View Timeline

- A 6-month timeline is automatically displayed.
- Each service is represented as a colored bar.
- **◀ ▶ buttons**: Navigate months
- **Right panel**: Shows total subscription cost for the current month

### Check Monthly Total

- Shows "Total subscription cost this month" below the timeline
- Sum of all active subscriptions

### Edit Service

1. Click the service name on the timeline.
2. Modify information in the right panel.
3. Click the **💾 Save** button.

### Delete Service

1. Select the service.
2. Click the **🗑️ Delete** button.
3. Confirm to delete.

## Features

| Feature | Description |
|---------|-------------|
| Timeline Dashboard | View 6 months of subscriptions at a glance |
| Monthly Total | Automatically calculates total subscription cost per month |
| Multiple Periods | Add multiple periods to one service (for breaks, etc.) |
| Color Customization | Choose color per service |
| Chrome Sync | Auto-sync across devices with Chrome account |

## Updates

- Chrome extensions update automatically.
- Manual: `chrome://extensions` → Click **Update**
