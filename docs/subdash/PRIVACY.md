# SubDash 개인정보 처리방침

최종 수정일: 2026-04-23

## 1. 수집하는 정보

SubDash는 서비스 제공을 위해 다음 정보를 처리합니다.

| 구분 | 항목 | 목적 | 저장 위치 |
|------|------|------|----------|
| 사용자 입력 | 구독 정보 (서비스명, 금액, 기간, 색상) | 구독 관리 | 사용자 기기 |
| 자동 수집 | 사용 통계 (추가/수정/삭제 이력) | 기능 개선 | 사용자 기기 |

- **개인 식별 정보(이름, 이메일, 계정 정보 등)는 수집하지 않습니다.**
- **금융 데이터(구독비 등)는 사용자 기기에만 저장되며, 외부로 전송되지 않습니다.**
- **구독 정보는 암호화되어 Chrome 동기화 저장소에 저장됩니다.**

## 2. 정보의 이용 목적

- 월별 구독 현황 타임라인 표시
- 구독 서비스 추가/수정/삭제 기능 제공
- 월별 총 구독비 자동 계산
- 기기 간 Chrome 동기화

## 3. 정보의 제3자 제공

수집된 정보는 원칙적으로 제3자에게 제공하지 않습니다. 다만 아래의 경우 예외로 합니다.

- 법령에 따른 요청이 있는 경우
- 사용자의 명시적 동의가 있는 경우

## 4. 정보의 보관 및 파기

- **로컬 저장소**: 사용자가 확장을 삭제할 때까지 기기에 보관
- **Chrome 동기화**: Chrome 계정으로 로그인한 다른 기기와 자동 동기화
- **자동 파기**: 확장 프로그램 삭제 시 저장된 모든 데이터 자동 삭제

## 5. 사용자의 권리

- **언제든지 확장 삭제**: 확장을 제거하면 모든 데이터가 자동으로 삭제됩니다
- **데이터 직접 관리**: 개별 구독 항목 수정/삭제 가능
- **동기화 제어**: Chrome 계정 동기화 설정으로 기기 간 동기화 제어 가능

## 6. 보안

- Chrome 확장의 sandbox 환경으로 보호됨
- Chrome Storage API를 통한 암호화된 저장소 사용
- 외부 서버로 데이터 전송 없음

## 7. 기술 정보

**저장 메커니즘**:
- Chrome Storage Sync: 구독 정보 저장 및 기기 간 동기화
- Chrome Storage Local: 로컬 캐시 데이터

**외부 통신**:
- 없음 (모든 데이터 로컬 처리)

## 8. 정책 변경

개인정보 처리방침은 서비스 개선을 위해 변경될 수 있습니다.
변경 시 확장의 업데이트 공지를 통해 안내됩니다.
