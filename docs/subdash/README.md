# SubDash

월별 구독 서비스 비용을 타임라인으로 시각적으로 관리하는 Chrome 확장입니다.

## 설치

1. 확장을 설치합니다.
2. 툴바에 SubDash 아이콘이 나타나면 설치 완료입니다.

## 사용법

### 서비스 추가

1. 툴바의 SubDash 아이콘을 클릭합니다.
2. **+ 추가** 버튼을 클릭합니다.
3. 오른쪽 패널에서 정보를 입력합니다:
   - 서비스명 (예: Claude Pro)
   - 월 금액 (₩ 단위)
   - 색상 선택
   - 구독 기간 (시작일 ~ 종료일)

### 타임라인 보기

- 6개월 타임라인이 자동으로 표시됩니다.
- 각 서비스가 색상 바로 표현됩니다.
- **◀ ▶ 버튼**: 월 이동
- **오른쪽 패널**: 현재 달의 총 구독비 표시

### 월별 합계 확인

- 타임라인 아래 "이번 달 전체 구독비" 표시
- 현재 진행 중인 모든 구독의 합계

### 서비스 수정

1. 타임라인의 서비스 이름을 클릭합니다.
2. 오른쪽 패널에서 정보를 수정합니다.
3. **💾 저장** 버튼을 클릭합니다.

### 서비스 삭제

1. 서비스를 선택합니다.
2. **🗑️ 삭제** 버튼을 클릭합니다.
3. 확인하면 삭제됩니다.

## 기능

| 기능 | 설명 |
|------|------|
| 타임라인 대시보드 | 6개월 구독 현황을 한눈에 보기 |
| 월별 합계 | 각 달의 총 구독비 자동 계산 |
| 여러 기간 설정 | 한 서비스에 여러 기간 추가 가능 (휴무 기간 제외 등) |
| 색상 커스터마이징 | 서비스별 색상 선택 |
| Chrome 동기화 | Chrome 계정으로 기기 간 자동 동기화 |

## 업데이트

- Chrome 확장 프로그램은 자동으로 업데이트됩니다.
- 수동: `chrome://extensions` → **업데이트** 클릭
