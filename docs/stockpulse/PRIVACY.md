# StockPulse 개인정보 처리방침

최종 수정일: 2026-04-23

## 1. 수집하는 정보

StockPulse는 서비스 제공을 위해 다음 정보를 처리합니다.

| 구분 | 항목 | 목적 | 저장 위치 |
|------|------|------|----------|
| 사용자 입력 | 주식 종목 코드 | 가격 추적 | 사용자 기기 |
| 자동 수집 | 알림 설정 (임계치, 간격) | 가격 변동 알림 | 사용자 기기 |
| 자동 수집 | 오류 로그 | 안정성 개선 | 사용자 기기 |

- **개인 식별 정보(이름, 이메일, 계정 정보 등)는 수집하지 않습니다.**
- **포트폴리오 데이터(주식 종목, 가격 등)는 사용자 기기에만 저장되며, 외부로 전송되지 않습니다.**

## 2. 외부 API 사용

### Yahoo Finance API (기본)
- **목적**: 주식 현재가, 변동률 조회
- **데이터 전송**: 조회하는 주식 종목 코드만 전송
- **개인정보 포함 여부**: 없음 (종목 코드만 전송)

### Alpha Vantage / Finnhub (백업)
- **목적**: Yahoo Finance 오류 시 대체 데이터 소스
- **데이터 전송**: 조회하는 주식 종목 코드만 전송
- **개인정보 포함 여부**: 없음

## 3. 정보의 이용 목적

- 주식 가격 실시간 조회
- 가격 변동 시 사용자 알림
- 서비스 오류 진단 및 안정성 개선

## 4. 정보의 보관 및 파기

- **로컬 저장소**: 사용자가 확장을 삭제할 때까지 기기에 보관
- **API 로그**: 수집일로부터 90일 후 자동 삭제
- **자동 파기**: 확장 프로그램 삭제 시 저장된 모든 데이터 자동 삭제

## 5. 사용자의 권리

- **언제든지 확장 삭제**: 확장을 제거하면 모든 데이터가 자동으로 삭제됩니다
- **종목 관리**: 추적하는 주식 종목 추가/제거 가능
- **알림 설정 제어**: 임계치 및 알림 간격 사용자 설정

## 6. 보안

- Chrome 확장의 sandbox 환경으로 보호됨
- HTTPS를 통한 API 통신 (암호화)
- 개인 포트폴리오 데이터는 로컬에만 저장

## 7. 기술 정보

**저장 메커니즘**:
- Chrome Storage: 주식 종목, 알림 설정
- IndexedDB: 가격 이력 데이터

**전송되는 데이터**:
- API 요청 시: 종목 코드, 현재 시간
- 분석 데이터: 없음

## 8. 정책 변경

개인정보 처리방침은 서비스 개선을 위해 변경될 수 있습니다.
변경 시 확장의 업데이트 공지를 통해 안내됩니다.
