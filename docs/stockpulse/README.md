# StockPulse

StockPulse는 실시간 주식 시세 및 알림을 제공하는 데스크톱/확장형 제품입니다.

- 제품명: StockPulse
- 최신 버전: (릴리스 노트 참조)

## 설치/업데이트 가이드

### 설치

1. [Chrome 웹 스토어](#)에서 "StockPulse"를 검색합니다.
2. **Chrome에 추가** 버튼을 클릭합니다.
3. 설치 완료 후 브라우저 툴바에서 StockPulse 아이콘을 확인합니다.

### 업데이트

- Chrome 확장 프로그램은 자동으로 업데이트됩니다.
- 수동 업데이트: `chrome://extensions` → 개발자 모드 → **업데이트** 클릭
