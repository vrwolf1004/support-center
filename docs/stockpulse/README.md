# StockPulse

관심 종목의 주가를 자동으로 확인하고, 가격 변동 시 알림을 보내주는 Chrome 확장 프로그램입니다.

## 설치

1. Chrome 웹 스토어에서 "StockPulse"를 검색합니다.
2. Chrome에 추가를 클릭합니다.
3. 툴바에 StockPulse 아이콘이 나타나면 설치 완료입니다.

## 사용법

### 종목 추가

1. 툴바의 StockPulse 아이콘을 클릭합니다.
2. **+** 버튼을 누릅니다.
3. 종목 코드를 입력합니다. (예: `005930.KS`, `AAPL`)
4. **Save**를 클릭하면 목록에 추가됩니다.

### 주가 확인

- 팝업을 열면 종목별 현재가, 변동폭, 등락률을 확인할 수 있습니다.
- **↻** 버튼으로 즉시 새로고침할 수 있습니다.
- 자동으로 일정 간격(기본 10분)마다 갱신됩니다.

### 알림

- 설정한 기준(기본 2%) 이상 가격이 변동하면 Chrome 알림이 표시됩니다.

### 목록 정렬

- 종목을 드래그하여 원하는 순서로 정렬할 수 있습니다.

## 설정 (Options)

팝업 하단의 **⚙ Options**를 클릭하면 설정 페이지가 열립니다.

| 항목 | 설명 | 기본값 |
|------|------|--------|
| 데이터 제공자 | Yahoo Finance, Finnhub, 공공데이터포털, iTick, FMP | Yahoo Finance |
| 알림 기준 (%) | 이 비율 이상 변동 시 알림 | 2% |
| 조회 간격 (분) | 자동 갱신 주기 | 10분 |
| 색상 모드 | 🇰🇷 한국식 (빨강↑/파랑↓) · 🌍 글로벌 (초록↑/빨강↓) | 한국식 |
| 언어 | 한국어, English, 日本語, Español, Русский | 한국어 |

### 데이터 제공자별 안내

| 제공자 | API 키 | 지원 시장 |
|--------|--------|-----------|
| Yahoo Finance | 불필요 | 전체 |
| Finnhub | 필요 | 미국/글로벌 |
| 공공데이터포털 | 필요 | 한국 |
| iTick | 필요 | 전체 |
| FMP | 필요 | 미국/글로벌 |

### 종목 백업/복원

- **Export**: 현재 종목 목록을 JSON 파일로 다운로드
- **Import**: JSON 파일에서 종목 목록을 불러오기

## 업데이트

- Chrome 확장 프로그램은 자동으로 업데이트됩니다.
- 수동: `chrome://extensions` → **업데이트** 클릭
