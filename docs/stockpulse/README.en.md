# StockPulse

A Chrome extension that automatically checks stock prices of your interest and sends notifications when prices change.

## Installation

1. Search for "StockPulse" on the Chrome Web Store.
2. Click "Add to Chrome".
3. Installation is complete when the StockPulse icon appears in the toolbar.

## Usage

### Add Stock

1. Click the StockPulse icon in the toolbar.
2. Click the **+** button.
3. Enter the stock code. (Example: `005930.KS`, `AAPL`)
4. Click **Save** to add to the list.

### Check Stock Price

- Open the popup to see the current price, change amount, and percent change for each stock.
- Click the **↻** button to refresh immediately.
- Updates automatically at regular intervals (default 10 minutes).

### Notifications

- A Chrome notification appears when the price changes by the threshold you set (default 2% or more).

### Sort List

- Drag stocks to arrange them in your preferred order.

## Updates

- Chrome extensions update automatically.
- Manual: `chrome://extensions` → Click **Update**
